# plausible_worlds
Here I keep the stuff going into the Plausible Worlds blog



# Repo map

my-blog-repo/
├── .gitignore             # Files to ignore (e.g., .Rproj.user, _site/)
├── _quarto.yml            # The main configuration (Navbar, project type, theme)
├── styles.css             # Your custom CSS for Corbel and Cambria fonts
├── index.qmd              # The Homepage (usually a listing of all latest posts)
├── about.qmd              # The "About" page
├── contact.qmd            # The "Contact" page (containing the Netlify form HTML)
├── rpgs.qmd               # Listing page specifically for 'RPGs' content
├── pseudo.qmd             # Listing page for 'Pseudo-Intellectualism' content
├── posts/                 # The folder where all your articles live
│   ├── _metadata.yml      # Default options for all posts (e.g., freeze settings)
│   ├── 2024-02-01-first-post/
│   │   ├── index.md       # The article text
│   │   └── image.jpg      # An image for the article
│   └── 2024-02-05-second-post/
│       └── index.md
└── images/                # Site-wide images (logo, profile pic, etc.)
